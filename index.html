How do I set it up?

  1. Make sure that you are running Google Chrome.
  2. Install User Agent Switcher and Manager from the Google Chrome Web Store.
  3. Open User Agent Switcher and Manager, select Chrome as the browser, and select either Android or iOS as the platform.
  4. Click on "Apply (active window)" or "Apply (all windows)" to activate the mobile user agent.

  5. Refresh this page. If you don't see this popup showing up again, then you are good to go!
